# Python Practice Projects

This repository contains a collection of beginner-to-intermediate Python programs developed while learning programming fundamentals.  
The goal of this repository is to practice problem solving, strengthen core Python concepts, and build small utility applications.

## Overview

The programs in this repository cover different areas of programming including:

- Mathematical algorithms
- Text processing
- Number analysis
- Simple games
- Utility applications

Each program focuses on improving logical thinking and understanding Python syntax and control structures.

## Projects Included

### Utility Programs
- Calculator
- Password Generator
- To-Do Manager
- Student Marks Manager

### Mathematical Programs
- Factorial Calculator
- Fibonacci Sequence Generator
- Prime Number Checker
- Largest of Three Numbers
- Sum of Numbers
- Multiplication Table Generator

### Text Processing
- Word Counter
- Vowel Counter
- String Reversal
- Palindrome Checker

### Mini Games
- Number Guessing Game

## Technologies Used

## How to Run the Programs

1. Install Python 3 on your system
2. Clone the repository

```
git clone https://github.com/yourusername/python-practice.git
```

3. Navigate to the project folder

```
cd python-practice
```

4. Run any program

```
python filename.py
```

Example:

```
python calculator.py
```
